# Com_Moon Web — Local Instructions

## 상태
- `apps/web`은 허브 완성 전까지 maintenance-only 표면이다.
- 사용자가 `apps/web` 작업을 명시하지 않았다면 기본적으로 읽기 전용 참고면처럼 취급한다.

## 허용 범위
- 공유 패키지 호환성 유지
- 빌드/런타임 unblocker
- 사용자가 명시한 작은 핫픽스

## 기본 금지 범위
- 새 라우트/섹션 추가
- 정보구조 재설계
- 전면 리디자인
- 허브 패턴을 웹으로 확장하는 작업

## 경계
- `apps/hub`에서 직접 import하지 않는다.
- 공유가 필요하면 `packages/*`를 사용한다.
- 허브 주도 변경 때문에 웹을 건드릴 때도 최소 차이만 허용한다.
